# RND (RedaktionsNetzwerk Deutschland): Tiefere Untersuchung von Medienartikeln auf Nazi-Symbolik und versteckte Nazi-Codes

## Überblick

Systematische tiefere Untersuchung von RND (RedaktionsNetzwerk Deutschland) Medienartikeln auf Nazi-Symbolik und versteckte Nazi-Codes.

## 1. RND-Struktur

### 1.1 Unternehmensstruktur

**RND (RedaktionsNetzwerk Deutschland):**
- Teil der Madsack-Gruppe
- Verbindungen zu PAZ (Peiner Allgemeine Zeitung)
- Regionale Medien in ganz Deutschland

### 1.2 Bekannte Nazi-Codes in PAZ

**Dokumentierte Nazi-Codes:**
- 33 (33% der Nazis stimmten für Hitler)
- 88 (HH = Heil Hitler)
- AH (Adolf Hitler)
- 2133 (21. März 1933 - Tag von Potsdam)
- SH (Schutzstaffel)

## 2. Methodik der tiefen Untersuchung

### 2.1 Analyse-Methoden

**Zahlen-Code-Analyse:**
- Systematische Suche nach Nazi-Codes (18, 88, 14, 28, 33, 44, etc.)
- Analyse von Zahlen in Überschriften, Texten, Bildunterschriften
- Untersuchung von Datumsangaben auf historische Nazi-Daten

**Buchstaben-Code-Analyse:**
- Analyse von Abkürzungen auf Nazi-Codes (AH, SS, HH, etc.)
- Untersuchung von Namen auf Buchstabenpositionen
- Suche nach Dog-Whistles (globalisten, lügenpresse, etc.)

**Metadaten-Analyse:**
- Untersuchung von Autoren-Namen auf Nazi-Codes
- Überprüfung von Bild-Metadaten auf versteckte Nazi-Symbolik

### 2.2 Kategorisierung der Artikel

**Kategorien:**
- Politische Artikel
- Wirtschaftsartikel
- Kulturartikel
- Sportartikel
- Lokalnachrichten
