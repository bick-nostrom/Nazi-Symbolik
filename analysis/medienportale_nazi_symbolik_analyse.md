# Medienportale Nazi-Symbolik Analyse

## Überblick

Analyse spezifischer Medienportale auf Nazi-Symbolik, mit Fokus auf Zahlen, Metadaten, visuelle Symbolik und sublimative Techniken.

## Methodik

**Untersuchungskriterien:**
- Zahlen-Codes in Artikeln und Überschriften
- Metadaten-Analyse (Timestamps, URLs, HTML)
- Visuelle Symbolik in Bildern und Videos
- Dog-Whistles in Texten
- Algorithmen-Optimierung
- Cross-Referenzierung

**Einschränkungen:**
- Öffentlich zugängliche Informationen
- Verifizierbare Daten
- Kontext-Abhängigkeit
- Falsch-Positive-Vermeidung

## Zusammenfassung

### Vorläufige Ergebnisse

**Alle Medienportale:**
- Keine offensichtlichen Nazi-Codes in den Namen
- Keine verifizierten Geburtsdaten (nicht relevant für Medienportale)
- Social Media Zahlen: Erfordern scraping
- Content-Zahlen: Erfordern scraping
- Metadaten: Erfordern scraping

### Empfehlungen

**Notwendige Schritte:**
1. Scraping der Websites (RTL, n-tv, Focus, Bild, Welt, Spiegel, FAZ, Tagesschau, ZDF, ARD)
2. Analyse der Artikel-Titel und Überschriften auf Nazi-Codes
3. Metadaten-Analyse (Timestamps, URLs, HTML)
4. Bildanalyse auf visuelle Symbolik
5. Textanalyse auf Dog-Whistles

**Tools:**
- `tools/web_scraper.py` für Website Scraping
- `tools/nazi_symbol_detector.py` für Textanalyse
- `tools/advanced_detector.py` für ML-gestützte Erkennung

### Limitationen

**Aktuelle Einschränkungen:**
- Kein Internetzugriff für Echtzeit-Scraping
- Keine konkreten Daten
- Kein Zugriff auf Bilder für visuelle Analyse
- Keine verifizierten Daten

**Benötigte Daten:**
- Konkrete URLs
- Artikel-Inhalte
- Bilder für visuelle Analyse
- Texte für Dog-Whistle-Analyse

### Hypothese

**Politische Ausrichtung und Nazi-Symbolik:**
- **Hypothese:** Medienportale mit konservativer Ausrichtung könnten höhere Raten an subtiler Nazi-Symbolik aufweisen
- **Begründung:** Konservative Medien könnten eher anfällig für Dog-Whistles und Relativierung sein
- **Gegenhypothese:** Öffentlich-rechtliche Medien könnten strengere Standards haben

**Erforderliche Validierung:**
- Empirische Datenerhebung
- Statistische Analyse
- Kontext-Prüfung
- Peer Review

*Diese Analyse basiert auf öffentlich zugänglichen Informationen. Für definitive Einschätzungen ist Scraping der Websites erforderlich.*
