# Influencer Nazi-Symbolik Analyse

## Überblick

Analyse spezifischer Influencer auf Nazi-Symbolik, mit Fokus auf Zahlen, Geburtsdaten und andere Datumsangaben.

## Methodik

**Untersuchungskriterien:**
- Zahlen-Codes in Namen, Biografien, Social Media
- Geburtsdaten und andere Datumsangaben
- Zahlen in Content (Captions, Video-Titel, etc.)
- Visuelle Symbolik (Outfits, Logos, Hintergründe)
- Dog-Whistles in Texten
- Metadaten-Analyse

**Einschränkungen:**
- Öffentlich zugängliche Informationen
- Verifizierbare Daten
- Kontext-Abhängigkeit
- Falsch-Positive-Vermeidung

### Empfehlungen

**Notwendige Schritte:**
1. Scraping der Social Media Kanäle (Instagram, YouTube, TikTok)
2. Analyse der Captions und Video-Titel auf Nazi-Codes
3. Bildanalyse auf visuelle Symbolik
4. Textanalyse auf Dog-Whistles
5. Metadaten-Analyse

**Tools:**
- `tools/web_scraper.py` für Social Media Scraping
- `tools/nazi_symbol_detector.py` für Textanalyse
- `tools/advanced_detector.py` für ML-gestützte Erkennung

### Limitationen

**Aktuelle Einschränkungen:**
- Kein Internetzugriff für Echtzeit-Scraping
- Keine verifizierten Geburtsdaten für die meisten Influencer
- Keine konkreten Social Media Zahlen
- Kein Zugriff auf Bilder und Videos

**Benötigte Daten:**
- Konkrete Social Media URLs
- Geburtsdaten (wenn verfügbar)
- Bilder für visuelle Analyse
- Texte für Dog-Whistle-Analyse
